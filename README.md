eva
===
Open up terminal and perform these cmds.

1. Install Meteor

    $ curl https://install.meteor.com | /bin/sh

2. Install Meteorite

    $ npm install -g meteorite

3. Clone this repo

    $ git clone git@github.com:loneleeandroo/eva.git

4. Go to repo

    $ cd eva

5. Start server

    $ mrt
